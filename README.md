# UW Capstone Mental Health Comorbidity

This repository contains code and project materials for a capstone project focused on pediatric mental health comorbidities and healthcare utilization.

## Project Overview
The project examines how mental health comorbidity patterns are associated with healthcare utilization outcomes using electronic health record data.

## Methods
- Data cleaning and preprocessing
- Exploratory data analysis
- Regression modeling
- Machine learning risk prediction

## Tools
- Python
- SQL
- XGBoost
- Statistical modeling

## Notes
Data are not publicly shared due to privacy restrictions.
